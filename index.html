GDT Pre-Filling System

Small-taxpayer self-declared backend support, delivered through a modern Angular + Spring Boot stack.

Angular Spring Boot Docker bitech

Big Picture

📱

Frontend (Web)

Angular/TypeScript application for tax office dashboards, reporting, and business detail workflows.

⚙️

Backend (API)

Spring Boot service exposing tax-domain APIs used by the web UI and downstream consumers.

🐳

Deployment

JAR built in IDE, copied to WEB-IWP, and deployed with Docker for repeatable delivery.

Modules & Screens

Summary Dashboard

Aggregate metrics, monthly charts, and KPI cards for tax offices.

Enterprise Registry

Paginated list of declared enterprises with filters and detail drill-downs.

Payment Report

Tax payment reporting by channel, status, and period.

Inventory Report

Government inventory and liability views.

Architecture

User

Tax officer or admin

→

Angular Web UI

Components, routing, state management, Khmer-enabled views

→

Spring Boot Backend

Tax APIs, domain services, JPA persistence

→

Database / Store

Enterprise, payment, inventory, and declaration data

Frontend and backend are packaged and deployed independently. The web UI is served statically, while the backend provides JSON APIs consumed by the UI.
