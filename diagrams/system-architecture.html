GDT Pre-Filling System — System Architecture

High-level flow: User → Angular Web UI → Spring Boot Backend → Docker on WEB-IWP

Frontend

  • Angular + TypeScript app
  • Tax office dashboards and reporting UI
  • Deployed separately from backend
  • Talks to backend via REST

Backend

  • Spring Boot 3.4.8
  • Runs on Java 21
  • Packaged as executable JAR
  • Serves JSON APIs to the UI

Deployment

  • Build JAR in IntelliJ IDEA
  • Copy JAR to WEB-IWP via SFTP
  • Run in Docker on port 8080
  • Verify at 10.255.255.6:19002
